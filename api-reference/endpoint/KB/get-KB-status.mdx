---
title: 'Get Knowledge Base Corpus Status'
api: 'GET /v1/knowledgeBase/corpusstatus/{projectId}'
description: "Get details of documents and paragraphs uploaded to the knowledge base."
---

### Authorizations

<ParamField header="x-api-key" type="string" required>
  API key associated with your project/workspace.
</ParamField>

---

### Path Parameters

<ParamField path="projectId" type="string" required>
  The UUID associated with the project/workspace.
</ParamField>

---

### Response

<ResponseField name="data" type="array">
An array of objects containing details of files uploaded to the project. Each object includes:
- **filename** (string): Name of the file.
- **state** (string): Status of the file (e.g., "finished").
- **eTag** (string): Unique identifier for the uploaded file.
- **location** (string): URL of the uploaded file.
- **uploadDate** (number): Timestamp of when the file was uploaded.
</ResponseField>

---
title: 'Correct Training Sample'
openapi: 'POST /v1/trainingsamples/correctresponse/{:projectId}/{:trainingId}'
description: "Update the response associated with a query in a specific training sample. Automatically sets the status to APPROVED."
---
### Authorizations
<ParamField header="x-api-key" type="string" required>
API key associated with your project/workspace
</ParamField>

### Path Parameters
<ParamField path="projectId" type="string" required>
The unique identifier of the project associated with the training sample.
</ParamField>
<ParamField path="trainingId" type="string" required>
The unique identifier of the training sample to be corrected.
</ParamField>



### Response

<ResponseField name="msg" type="object">
Confirmation message indicating whether the document updated succesfully

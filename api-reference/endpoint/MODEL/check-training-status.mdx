---
title: 'Check training Status'
openapi: 'PUT /v1/finetuning/checkTrainingStatus/{projectId}/{modelId}'
description: "Check the current fine tuning status of your model. The status only updates when you call this API."
---
### Authorizations
<ParamField header="x-api-key" type="string" required>
The API key used for authentication and authorization purposes.
</ParamField>

### Path Parameters
<ParamField path="projectId" type="string" required>
The unique identifier of the project to which the model belongs.
</ParamField>
<ParamField path="modelId" type="string" required>
The unique identifier of the model for which the training status is being checked.
</ParamField>

___

```

```
</ResponseField>

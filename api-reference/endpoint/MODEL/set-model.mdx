---
title: 'Set Model'
openapi: 'PUT /v1/finetuning/checkTrainingStatus/{:projectId}/{:modelId}'
description: "Set a model to be used for responding to queries for your workspace."
---
### Authorizations
<ParamField header="x-api-key" type="string" required>
API key associated with your project/workspace
</ParamField>

### Path Parameters
<ParamField path="projectId" type="string" required>
The unique identifier of the project associated with the model for which the training status is being checked.
</ParamField>
<ParamField path="modelId" type="string" required>
The unique identifier of the model for which the training status is being checked.
</ParamField>

### Response
___
<ResponseField name="msg" type="object">
Confirmation message indicating whether the model was successfully set or not
</ResponseField>

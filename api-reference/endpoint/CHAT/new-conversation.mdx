title: 'Start a new conversation'
api: 'POST v1/chatbot'
description: "Use this API endpoint to generate a response from the LLM, tailored to your custom knowledge base and the specified prompt."
---
### Authorizations
<ParamField header="x-api-key" type="string" required>
API key associated with your project/workspace
</ParamField>


### Body Parameters
___
<ParamField body="projectId" type="string" required>
The UUID associated with a project/workspace.
</ParamField>
<ParamField body="question" type="string" required>
The question to be submitted to the LLM
</ParamField>

### Response
___
<ResponseField name="response" type="string">
 The response generated by the LLM.
</ResponseField>
<ResponseField name="conversationId" type="string">
The ID associated with the conversation.
</ResponseField>

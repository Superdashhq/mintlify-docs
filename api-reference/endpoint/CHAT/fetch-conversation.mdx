---
title: 'Fetch conversation(s)'
api: 'GET v1/conversation/project/{:projectId}/{:userIdentifier}'
description: "Use this API endpoint to retrieve the complete conversation linked to a specific user identifier."
---
### Authorizations
<ParamField header="x-api-key" type="string" required>
API key associated with your project/workspace
</ParamField>


### Body Parameters
___
<ParamField body="userIdentifier" type="string" required>
Unique customerID used to store the customer conversation.
</ParamField>
<ParamField body="projectId" type="string" required>
The UUID associated with a project/workspace.
</ParamField>
<ParamField body="question" type="string" required>
The question to be submitted to the LLM
</ParamField>

### Response
___
<ResponseField name="project" type="string">
UUID associated with a project
</ResponseField>
<ResponseField name="_id" type="string">
The ID associated with the conversation.
</ResponseField>
<ResponseField name="messages" type="array">
Array containing conversation threads associated with the user identifier
</ResponseField>
